# Phylogeny-blaOXA23-Project

This repository contains the full work behind a pre-graduate research project on the phylogenetic analysis of the *bla*OXA-23 resistance gene. It spans two distinct but complementary worlds: the biological research done by hand, and a custom web application built to support and present it.

---

## Repository Structure

```
Phylogeny-blaOXA23-Project/
├── biology-research/       # All biological research work done by hand
└── system-impl/            # Web application built to support the research
```

### `biology-research/`

This folder contains everything produced during the pre-graduate research project — sequence alignments, phylogenetic trees, analysis results, scripts, and supporting files, all worked through manually as part of the research process.

It will also include a PDF titled **"blaoxa-23 research paper"**, which contains the complete thesis for this project.

### `system-impl/`

This folder holds the personal implementation of the web application developed alongside the research. The app was built to manage, visualize, and interact with the data produced during the biological analysis.

---

## Documentation

Each module within the project has its own README explaining its specific purpose, structure, and usage. The `system-impl/` folder in particular contains a top-level README that describes the entire system architecture and how to run it.
